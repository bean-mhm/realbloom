This folder contains the internal OpenColorIO config for RealBloom, which is a custom config made by combining Troy Sobotka's new AgX and Eary Chow's version of AgX.

You can change the user config in "[Program Directory]/ocio".
Do not modify the contents of this folder unless you know what you're doing.
