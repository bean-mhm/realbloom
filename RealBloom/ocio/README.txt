This folder contains the default OpenColorIO config for RealBloom, which is a custom config made by combining Eary's AgX and Troy Sobotka's new AgX config generator.

You are free to swap this with your own custom OCIO config.

Troy's new AgX:
https://github.com/sobotka/SB2383-Configuration-Generation

Eary's AgX:
https://github.com/EaryChow/AgX/releases/tag/v11.9
